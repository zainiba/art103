Art 106 - Human Machine Interface

Spring 2020

Proposal

For the final project I will be working with Talea to improve the software of the emotional support device. Currently it is a prototype of a comforting long distance communication system. We want to create a device that people will use for emotional support especially in these times where people are feeling isolated. I want to add drawings to the led matrix as well as change the phrases to be more supportive rather than filled with longing. For the speaker, I want to add a short melody for each phrase that shows up on the led matrix. I also want to speed up how fast the sensor reads temperatures.
